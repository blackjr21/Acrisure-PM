# TIER 4: FORWARD PLANNING (Optional)

**Agent:** meeting-agenda-preparer  
**Purpose:** Prepare agendas for upcoming meetings and close completed ones

**Note:** This tier is optional.

---

## Agent Delegation
```
DELEGATE to meeting-agenda-preparer agent (MUST BE USED for agenda preparation):

Task: Prepare agendas for upcoming recurring meetings based on {Month} history

Requirements:
1. Identify recurring meetings
2. Generate context-rich agendas with 9 sections:
   - Meeting metadata
   - Attendee context
   - Project context
   - Previous action items
   - Long-running topics
   - Deliverables/milestones
   - New discussion topics
   - Decisions needed
   - Next steps
3. Mark completed meetings as [Closed]

Success Criteria:
- Each recurring meeting has agenda
- All 9 sections present
- Action statuses current
- Old meetings marked [Closed]

Estimated time: 15-20 minutes
```
