# Resources

This folder contains templates and references for the Claritev PM repository.

---

## Templates

Located in `Resources/templates/`:

- **todos-template.md** - Template for tracking tasks at program/project level
- **plan-template.md** - Template for project planning documents
- **status-template.md** - Template for weekly/monthly status updates
- **meeting-notes-template.md** - Template for meeting notes
- **transcription-template.md** - Template for meeting transcriptions

---

## How to Use Templates

### For New Projects:
1. Copy the templates from `Resources/templates/` to your project folder
2. Rename them (e.g., `todos-template.md` â†’ `todos.md`)
3. Fill in the project-specific details

### For New Meetings:
1. Create a new folder: `Meetings/[Type]/YYYY-MM-DD-Topic/`
2. Copy `transcription-template.md` and `meeting-notes-template.md`
3. Rename and fill in meeting details
